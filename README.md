## 技術：
  - **後端：** Spring Boot
  - **前端：** Alpine.js
  - **語言：** Java 17, JavaScript
  - **資料庫：** MongoDB, Neo4j
  - **雲端部署：** Google Cloud
  - **第三方 API：** openAI (gpt-4o-mini, text-embedding-3-small)
  - **即時通訊：** WebSocket
